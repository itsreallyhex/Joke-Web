# Joke-Web

صاحبي قال نكتة سيئة. قررت أخلده على الإنترنت.

موقع بسيط مكوّن من HTML وCSS خالص — بنيته كمشروع تعلم
وكرد مناسب على ما صدر منه.

## الصفحات

| الملف | المحتوى |
|---|---|
| `index.html` | الصفحة الرئيسية |
| `abdullah.html` | كل اللي تحتاج تعرفه عن عبدالله |
| `Comingsoon.html` | قيد التطوير |
| `style.css` | التصميم |
| `assets/` | الصور |

## التشغيل

افتح `index.html` في المتصفح. لا يلزم شيء ثاني.

## ملاحظة

الموقع يدعم العربية واتجاه RTL.
متوافق مع الجوال.
