# Joke-Web

عبدالله قال نكتة سيئة. قررت اسوي موقع كامل.

موقع بسيط مكوّن من HTML وCSS  — بنيته كمشروع تعلم
وكرد مناسب على ما صدر من عبدالله.

## الصفحات

| الملف | المحتوى |
|---|---|
| `index.html` | الصفحة الرئيسية |
| `abdullah.html` | مقال عن العفويه وصور لعبدالله|
| `Comingsoon.html` | قيد التطوير |
| `style.css` | التصميم |
| `assets/` | الصور |

> تنويه انه كل المعلومات من المقال لا امثل باي شكل عبدالله. مافي اي معلومه شخصيه عنه، جميعها مجرد صور لمحادثات عامه.
## التشغيل

افتح `index.html` في المتصفح. ماتحتاج شيء ثاني.

## ملاحظة

الموقع يدعم العربية واتجاه RTL.
متوافق مع الجوال.
